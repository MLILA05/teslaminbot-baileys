# <div align='center'>Dml-Baileys</div>

Dml-Baileys is a WebSockets-based TypeScript/Javascript library for interacting with the WhatsApp Web API.


> You might want to use **[Dml-baileys@1.0.0](https://github.com/DAUDY17/minbotv1)** which has the latest fixes and improvements.
